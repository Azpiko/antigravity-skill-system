# Blueprint: SaaS & Backoffice

Archetype pour outils de gestion et tableaux de bord data-centric.

## Objectifs
- **Data Management**: CRUD avancés, filtres, exports.
- **Sécurité**: Validation Zod multi-niveaux.
- **Productivité**: Raccourcis clavier, feedback immédiat, Skeletons.

## Structure Technique
- **Next.js**: Server Components (lecture), Client Components (formulaires).
- **Layout**: Sidebar persistante, navigation claire.
- **Auth**: Flux standardisé.

## Stack Spécifique
- **Forms**: React Hook Form + Zod.
- **Tables**: Pagination, tri, virtualisation.
- **Analytics**: Suivi actions.

## Checklist SAAS
- [ ] Accès sécurisés.
- [ ] Formulaires gérant loading/success/error.
- [ ] Navigation performante avec data.
